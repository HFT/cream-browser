Functions

paste (primary)
Paste URL. Open in the current tab the yanked URL.

Parameters

  • primary: true to paste the yanked url from the primary clipboard, false otherwise. (default: true)
yank (primary, selection)
Yank URL Copy current URL in the clipboard.

Parameters

  • primary: true to yank the current url or selection to the primary clipboard, false otherwise. (default: true)
  • selection: true to yank the current selection, false for the URL. (default: false)
